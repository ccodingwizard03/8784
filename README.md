## Installation

```bash
$ npm install
```

## Running the app

```bash
# after start scraping process
1. copy.env.example file
2. rename copied .env.example to .env
3. set your credentials in .env file
4. from app.ts file set your instagram login and password

# build the project
$ npm run build

# dev mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
