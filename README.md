## Installation

```bash
$ git clone https://github.com/ccodingwizard03/8784.git
$ cd 8784
$ npm install
```

## Config

After starting
1. copy .env.example file
2. rename copied .env.example to .env
3. set your credentials in .env file
4. from app.ts file set your instagram login and password

## Running the app

```bash
# build the project
$ npm run build

# dev mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
